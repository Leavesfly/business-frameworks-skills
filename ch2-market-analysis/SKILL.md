## 框架清单（09–21）

### ⑨ PEST 分析
- **用途**：扫描宏观环境。
- **维度**：**P**olitical 政治 / **E**conomic 经济 / **S**ocial 社会 / **T**echnology 技术
- **模板**：
  ```
  | 维度 | 机会 (Opportunity) | 威胁 (Threat) |
  |------|-------------------|---------------|
  | P    |                   |               |
  | E    |                   |               |
  | S    |                   |               |
  | T    |                   |               |
  ```
- **英文关键词**：PEST, PESTEL, macro environment

### ⑩ 五力分析（Porter's Five Forces）
- **用途**：判断行业吸引力与利润空间。
- **五力**：现有竞争者、潜在进入者、替代品、供应商议价能力、买方议价能力
- **模板**：
  ```
                  潜在进入者
                      │
                      ▼
    供应商  →  现有竞争者  ←  买方
                      ▲
                      │
                   替代品
  ```
  每一力：**强度（高/中/低）+ 关键驱动因素 + 应对**
- **英文关键词**：five forces, Porter, industry analysis

### ⑪ VRIO 分析
- **用途**：判断一项资源是否构成**持久竞争优势**。
- **四问**：Valuable? Rare? Inimitable? Organized?
- **模板**：
  ```
  | 资源 | V | R | I | O | 结论             |
  |------|---|---|---|---|------------------|
  |      | ✔ | ✔ | ✘ | ✔ | 暂时优势         |
  ```
- **英文关键词**：VRIO, resource based view, RBV

### ⑫ SWOT 分析
- **用途**：内外部因素交叉盘点。
- **模板**：
  ```
  ┌───────────┬───────────┐
  │ S 优势    │ W 劣势    │
  ├───────────┼───────────┤
  │ O 机会    │ T 威胁    │
  └───────────┴───────────┘
  ```
- 升级用法：见第 4 章 `交叉 SWOT`㉞
- **英文关键词**：SWOT, strengths weaknesses opportunities threats

### ⑬ 帕雷托分析（Pareto / 80-20）
- **用途**：找出"造成 80% 结果的那 20% 关键因素"。
- **步骤**：
  1. 列出所有因素及其数值（金额/次数）
  2. 降序排列，计算累计占比
  3. 锁定累计占比 ≤ 80% 的头部因素
- **模板**：
  ```
  | 因素 | 数值 | 占比% | 累计% |
  |------|------|-------|-------|
  ```
- **英文关键词**：Pareto, 80/20 rule, ABC analysis

### ⑭ RFM 分析
- **用途**：对已有用户分层，找出"高价值 / 流失风险 / 沉睡唤醒"群体。
- **三维**：
  - **R**ecency：最近一次购买距今的时间（越近得分越高）
  - **F**requency：近期购买频次（越高越好）
  - **M**onetary：近期累计消费金额（越高越好）
- **打分法**：每维分 1–5 分 → 形成 555 出头、111 沉睡的 125 个人群。
- **模板**：
  ```
  | 用户分群 | R | F | M | 语义标签    | 运营动作      |
  |---------|---|---|---|-----------|---------------|
  | 555出头  | 5 | 5 | 5 | 核心客户    | 专享权益 / VIP|
  | 511–155 | ↑ |↓↓| ↓ | 流失风险    | 召回 / 活动    |
  | 5↓↓     |↓↓| ↓ | ↓ | 新用户      | 新手引导      |
  | 1↓↓     |↓↓|↓↓|↓↓| 沉睡用户    | 唤醒 / 省成本|
  ```
- **英文关键词**：RFM, customer segmentation, CLV, cohort analysis

### ⑮ 人物画像（Persona）
- **用途**：把目标用户具象化为一个"人"。
- **模板**：
  ```
  姓名：          年龄：          职业：
  生活状态：
  目标 / 需求：
  痛点 / 焦虑：
  信息渠道：
  一句话：        （用他自己的话）
  ```
- **英文关键词**：persona, user profile, buyer persona

### ⑯ 同理心地图（Empathy Map）
- **用途**：从 6 个角度体会用户。
- **模板**：
  ```
  ┌──────────┬──────────┐
  │ 说 Says  │ 想 Thinks│
  ├──────────┼──────────┤
  │ 做 Does  │ 感 Feels │
  └──────────┴──────────┘
   痛点 Pains   ┆  收获 Gains
  ```
- **英文关键词**：empathy map, says thinks does feels

### ⑰ 客户体验旅程图（Customer Journey Map）
- **用途**：还原用户从认知到忠诚的全路径体验。
- **模板**：
  ```
  | 阶段     | 认知 | 考虑 | 购买 | 使用 | 忠诚 |
  |----------|------|------|------|------|------|
  | 行为     |      |      |      |      |      |
  | 想法     |      |      |      |      |      |
  | 情绪↑↓   |      |      |      |      |      |
  | 触点     |      |      |      |      |      |
  | 痛点/机会|      |      |      |      |      |
  ```
- **英文关键词**：customer journey map, touchpoint, moment of truth

### ⑱ 4P 分析
- **用途**：营销组合基础。
- **模板**：Product / Price / Place / Promotion，逐项列出当前策略与问题。
- **英文关键词**：4P, marketing mix

### ⑲ 4P + 提供内容与对象分析
- **用途**：4P 的升级版，加入 **Who（对象）× What（内容）**，确保"卖给谁 / 卖什么"与 4P 前后一致。
- **模板**：
  ```
  | Who 对象 | What 提供内容 | Product | Price | Place | Promotion |
  |----------|---------------|---------|-------|-------|-----------|
  |          |               |         |       |       |           |
  ```
- **扩展**：服务业可再升级为 7P（People / Process / Physical evidence）。
- **英文关键词**：4P extended, 7P service marketing, target-offering fit

### ⑳ 价值链分析（Value Chain）
- **用途**：拆解企业内部活动，找价值创造点。
- **结构**：
  - 主要活动：进货物流 → 生产 → 出货物流 → 市场营销 → 服务
  - 支持活动：基础设施 / 人力资源 / 技术开发 / 采购
- **模板**：对每个环节打分（成本 / 差异化贡献），找优化点。
- **英文关键词**：value chain, primary activities, support activities

### ㉑ 核心竞争力分析（Core Competence）
- **用途**：识别"别人学不会 + 能进入多个市场 + 客户感知到价值"的能力。
- **三问**：
  1. 对客户价值是否显著？
  2. 是否难以模仿？
  3. 能否迁移到其他业务？
- **英文关键词**：core competence, Hamel Prahalad

## 常用组合

- **进入新市场** → `PEST`⑨ + `五力`⑩ + `SWOT`⑫
- **盘点自身** → `VRIO`⑪ + `价值链`⑳ + `核心竞争力`㉑
- **读懂用户** → `人物画像`⑮ + `同理心地图`⑯ + `客户体验旅程图`⑰
- **找增长点** → `帕雷托`⑬ + `RFM`⑭
- **营销诊断** → `4P`⑱ → `4P+`⑲
